folkabulary
===========

Inspired by "The Largest Vocabulary in Hip Hip" by Matt Daniels. Here's our folky take on it.


# Setup

Clone this repo.

Change into the directory and run

    python -m SimpleHTTPServer

And go to

    localhost:8000
